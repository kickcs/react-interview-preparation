Опишите discriminated union `Shape = Circle | Square | Triangle`, где каждая фигура имеет поле `kind` с уникальным строковым значением. Затем напишите type guard `isCircle(shape: Shape): shape is Circle`, который возвращает `true` только для круга.

<StarterCode>

```ts
type Circle   = { kind: "circle";   radius: number };
type Square   = { kind: "square";   side: number };
type Triangle = { kind: "triangle"; base: number; height: number };

type Shape = Circle | Square | Triangle;

function isCircle(shape: Shape) {
  // Ваш код здесь
}

// ✅ Проверка
console.log(isCircle({ kind: "circle",   radius: 5 }));    // true
console.log(isCircle({ kind: "square",   side: 3 }));      // false
console.log(isCircle({ kind: "triangle", base: 4, height: 6 })); // false

// Проверка сужения типа
function getArea(shape: Shape): number {
  if (isCircle(shape)) {
    return Math.PI * shape.radius ** 2; // shape здесь Circle
  }
  return 0;
}
```

</StarterCode>

<Hint title="Подсказка 1">
Type guard — это функция, возвращаемый тип которой записывается как `param is Type`. Внутри можно использовать любое булево выражение. После вызова такой функции TypeScript знает, какой именно тип имеет значение.
</Hint>

<Hint title="Подсказка 2">
В discriminated union достаточно проверить дискриминирующее поле: `return shape.kind === "circle"`. Возвращаемый тип функции должен быть `shape is Circle`, чтобы TypeScript сузил тип внутри `if`-блока.
</Hint>

<Solution>

```ts
type Circle   = { kind: "circle";   radius: number };
type Square   = { kind: "square";   side: number };
type Triangle = { kind: "triangle"; base: number; height: number };

type Shape = Circle | Square | Triangle;

function isCircle(shape: Shape): shape is Circle {
  return shape.kind === "circle";
}

// ✅ Проверка
console.log(isCircle({ kind: "circle",   radius: 5 }));           // true
console.log(isCircle({ kind: "square",   side: 3 }));             // false
console.log(isCircle({ kind: "triangle", base: 4, height: 6 })); // false

function getArea(shape: Shape): number {
  if (isCircle(shape)) {
    return Math.PI * shape.radius ** 2; // shape здесь Circle
  }
  return 0;
}
```

</Solution>
