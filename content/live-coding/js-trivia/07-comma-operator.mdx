Оператор запятой — один из самых малоизвестных операторов в JavaScript. Он вычисляет каждый из своих операндов слева направо и возвращает значение **последнего** операнда.

**Предскажите, что выведет каждый `console.log`:**

<StarterCode>

```js
console.log((1, 2, 3));
console.log((true, false, true));

let x = (10, 20, 30);
console.log(x);

let a = 0;
let b = (a++, a++, a++);
console.log(a);
console.log(b);
```

</StarterCode>

<Hint title="Подсказка">
Оператор запятой вычисляет все операнды и возвращает результат последнего. Скобки обязательны при использовании вне `for`, чтобы отделить оператор от синтаксиса разделения аргументов/элементов. Помните, что `a++` возвращает значение **до** инкремента.
</Hint>

<Solution>

Вывод:
```
3
true
30
3
2
```

**Объяснение:**

- `(1, 2, 3)` → `3`: вычисляются все три выражения, возвращается последнее — `3`.

- `(true, false, true)` → `true`: аналогично, возвращается последний операнд — `true`.

- `let x = (10, 20, 30)` → `x = 30`: все три числа вычисляются, присваивается последнее.

- `let b = (a++, a++, a++)`:
  - Первый `a++`: `a` равно `0`, возвращает `0`, `a` становится `1`.
  - Второй `a++`: `a` равно `1`, возвращает `1`, `a` становится `2`.
  - Третий `a++`: `a` равно `2`, возвращает `2`, `a` становится `3`.
  - Оператор запятой возвращает результат последнего операнда — `2`.
  - Итого: `a = 3`, `b = 2`.

Оператор запятой чаще всего встречается в `for`-циклах для выполнения нескольких действий в секции инициализации или обновления: `for (let i = 0, j = 10; i < j; i++, j--)`.

</Solution>
