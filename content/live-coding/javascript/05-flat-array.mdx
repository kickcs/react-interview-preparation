Реализуйте функцию `flatten(arr)`, которая рекурсивно выравнивает произвольно вложенный массив в одномерный. Использовать встроенный `Array.prototype.flat()` запрещено.

<StarterCode>

```js
function flatten(arr) {
  // Ваш код здесь
}

// ✅ Проверка
const nested = [1, [2, [3, [4]], 5]];
const result = flatten(nested);

console.log(result.length === 5);                        // true
console.log(result[0] === 1);                            // true
console.log(result[3] === 4);                            // true
console.log(JSON.stringify(result) === '[1,2,3,4,5]');   // true
```

</StarterCode>

<Hint title="Подсказка 1">
Пройдитесь по каждому элементу массива. Если элемент сам является массивом — рекурсивно применяйте к нему `flatten`. Если нет — добавляйте его в результат.
</Hint>

<Hint title="Подсказка 2">
Удобный способ — использовать `reduce`: начальное значение `[]`, и для каждого элемента делать `concat` с результатом рекурсивного вызова. Либо использовать spread с рекурсией.
</Hint>

<Solution>

```js
function flatten(arr) {
  return arr.reduce((acc, item) => {
    if (Array.isArray(item)) {
      return acc.concat(flatten(item));
    }
    return acc.concat(item);
  }, []);
}
```

</Solution>
