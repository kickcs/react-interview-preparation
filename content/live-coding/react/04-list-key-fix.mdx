---
title: "Баг с key={index} в списке"
order: 4
---

Компонент рендерит список текстовых инпутов. Ключи заданы через `key={index}`. Введите текст в несколько полей, затем удалите элемент из середины — содержимое инпутов «съедет». Исправьте баг, добавив стабильные уникальные ключи.

<StarterCode>

```jsx
import { useState } from 'react';

const initialItems = [
  { id: 1, label: 'Первый' },
  { id: 2, label: 'Второй' },
  { id: 3, label: 'Третий' },
  { id: 4, label: 'Четвёртый' },
];

// ❌ key={index} — приводит к багу при удалении
export default function App() {
  const [items, setItems] = useState(initialItems);

  const remove = (index) => {
    setItems(prev => prev.filter((_, i) => i !== index));
  };

  return (
    <ul>
      {items.map((item, index) => (
        <li key={index}>
          <input placeholder={item.label} />
          <button onClick={() => remove(index)}>Удалить</button>
        </li>
      ))}
    </ul>
  );
}
```

</StarterCode>

<Hint title="Подсказка 1">
React использует `key` для сопоставления элементов при обновлении DOM. Если ключ — это индекс, то после удаления элемента из середины оставшиеся элементы получают чужие ключи, и React неверно переиспользует DOM-узлы.
</Hint>

<Hint title="Подсказка 2">
Замените `key={index}` на `key={item.id}`. Теперь у каждого элемента стабильный ключ, привязанный к данным, а не к позиции в массиве. Функцию `remove` можно переписать через `item.id` вместо индекса.
</Hint>

<Solution>

```jsx
import { useState } from 'react';

const initialItems = [
  { id: 1, label: 'Первый' },
  { id: 2, label: 'Второй' },
  { id: 3, label: 'Третий' },
  { id: 4, label: 'Четвёртый' },
];

// ✅ key={item.id} — стабильный уникальный ключ
export default function App() {
  const [items, setItems] = useState(initialItems);

  const remove = (id) => {
    setItems(prev => prev.filter(item => item.id !== id));
  };

  return (
    <ul>
      {items.map((item) => (
        <li key={item.id}>
          <input placeholder={item.label} />
          <button onClick={() => remove(item.id)}>Удалить</button>
        </li>
      ))}
    </ul>
  );
}
```

</Solution>
