---
title: "Что такое семантический HTML?"
order: 1
---

<AnswerGroup>
<Answer lang="en">

**Semantic HTML** means using HTML elements that clearly describe their meaning to both the browser and the developer. Instead of using generic `<div>` and `<span>` tags for everything, you use elements like `<header>`, `<nav>`, `<main>`, `<article>`, and `<footer>`.

Semantic elements improve **accessibility** because screen readers can understand the page structure. They also help with **SEO** since search engines can better identify the content.

Key semantic elements include: `<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<aside>`, `<footer>`, `<figure>`, and `<figcaption>`.

</Answer>
<Answer lang="ru">

**Семантический HTML** означает использование HTML-элементов, которые ясно описывают своё назначение как для браузера, так и для разработчика. Вместо того чтобы использовать универсальные теги `<div>` и `<span>` для всего подряд, вы применяете элементы вроде `<header>`, `<nav>`, `<main>`, `<article>` и `<footer>`.

Семантические элементы улучшают **доступность**, потому что программы чтения с экрана могут понять структуру страницы. Они также помогают с **SEO**, так как поисковые системы лучше распознают содержимое.

Основные семантические элементы: `<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<aside>`, `<footer>`, `<figure>` и `<figcaption>`.

</Answer>
</AnswerGroup>

```html
<header>
  <nav>
    <a href="/">Home</a>
    <a href="/about">About</a>
  </nav>
</header>

<main>
  <article>
    <h1>Understanding Semantic HTML</h1>
    <p>Semantic elements give meaning to your markup.</p>
  </article>

  <aside>
    <h2>Related Topics</h2>
    <p>CSS, Accessibility, SEO</p>
  </aside>
</main>

<footer>
  <p>&copy; 2024 My Website</p>
</footer>
```
