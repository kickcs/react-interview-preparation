Хочу создать простую платформу для подготовки к интервью для React разработчиков,
суть будет заключаться в том, что русскоговорящие люди будут готовиться к интервью на английском.
Мы будет предоставлять вопрос и давать ответ сразу на двух языках.
Мы должны будет приводить примеры кода, там где надо, чтобы человек мог понять, как решить задачу.
Ответы должны быть на максимально понятном английском, уровень примерно B2.

Структура приложения будет максимально проста, мы делим вопросы на категории и предоставляем ответы сразу на двух языках.
Слева будет sidebar с категориями, а справа - вопрос и ответ. Категории должны быть максимально широкими, чтобы охватывать разные типы вопросов. Вопросы должны иметь логическую цепочку, а не в рандомном порядке.

Используй bun, с помощью context7 найди самые актуальные версии библиотек и инструментов, вот список того, что ты можешь использовать если тебе это нужно.

Проработай дизайн с помощью frontend-design skill, используя библиотеку компонентов shadcn/ui и его skill.

Next.js 16 + TypeScript + Tailwind v4 + shadcn/ui + MDX + Bun + FSD. 

Если что-то не нужно так и говори, мы не будем использовать это.
