# UI Redesign — Спецификация

## Обзор

Переработка UI отображения вопросов: горизонтальный двуязычный лейаут, спойлер-эффект для ответов, единый шрифт Inter, код вынесен в отдельную секцию ниже ответов, увеличенные отступы.

## Изменения

### 1. Шрифт Inter

Заменить дефолтный шрифт на **Inter** (Google Fonts / `next/font/google`). Inter хорошо отображает и латиницу, и кириллицу с одинаковым визуальным весом. Для кода — **JetBrains Mono** (моноширинный).

**Файлы:** `src/app/layout.tsx`, `src/app/globals.css`

### 2. Горизонтальный лейаут ответов

EN и RU ответы располагаются параллельно в два столбца (CSS Grid `1fr 1fr`, gap `20px`). Оба блока содержат одинаковый по смыслу текст — EN с английским объяснением, RU с русским. Код НЕ входит ни в один из блоков.

На мобильных (<768px) — вертикальный стек (одна колонка).

**Файлы:** `src/shared/ui/answer.tsx` — изменить, чтобы два `<Answer>` компонента ложились в grid. Новый враппер-компонент `<AnswerGroup>` для grid-контейнера.

### 3. Спойлер-эффект

При открытии вопроса ответы скрыты. Видна только кнопка «Показать ответ / Show answer». Один клик раскрывает всё (оба ответа + код) с плавной анимацией.

Спойлер управляется на уровне `QuestionView` — оборачивает весь MDX-контент. Не требует изменений в MDX-файлах.

**Компонент:** Новый `src/shared/ui/spoiler.tsx` — клиентский компонент с `useState` для toggle. Анимация через CSS `max-height` + `opacity` transition.

**Файлы:** `src/shared/ui/spoiler.tsx` (новый), `src/widgets/question-view/ui/question-view.tsx` (обернуть MDX в Spoiler)

### 4. Код вынесен ниже ответов

Секция кода располагается после обоих ответов (вне grid), с зелёным бейджем `CODE` и указанием языка. Код отделён от текстовых блоков отступом `24px`.

**Изменение формата MDX:**

Было:
```mdx
<Answer lang="en">
Текст + код внутри
</Answer>
<Answer lang="ru">
Только текст
</Answer>
```

Стало:
```mdx
<AnswerGroup>
<Answer lang="en">
Текст (без кода)
</Answer>
<Answer lang="ru">
Текст (идентичный по содержанию)
</Answer>
</AnswerGroup>

```jsx
// код отдельно
```
```

Код остаётся обычным markdown code block — `rehype-pretty-code` подсвечивает его как раньше. Никакого специального компонента для кода не нужно.

**Файлы:** Все `content/**/*.mdx` файлы (13 штук), `src/shared/ui/answer-group.tsx` (новый)

### 5. Увеличенные отступы

- Между колонками EN/RU: `gap-5` (20px)
- Между AnswerGroup и секцией кода: `mt-6` (24px) — стилизуется через глобальные prose-стили
- Между секцией кода и навигацией: `mt-8` (32px)
- `max-width` контента увеличен с `720px` до `900px` для двух колонок

**Файлы:** `src/widgets/question-view/ui/question-view.tsx`, `src/app/globals.css`

### 6. Одинаковое содержание EN и RU

Оба ответа должны содержать одинаковую информацию — только на разных языках. Русский блок больше НЕ урезан по сравнению с английским. Код не дублируется — он в отдельной секции.

**Файлы:** Все `content/**/*.mdx` файлы (13 штук)

## Новые компоненты

### `<AnswerGroup>`
- Grid-контейнер: `grid grid-cols-1 md:grid-cols-2 gap-5`
- Оборачивает два `<Answer>` компонента
- MDX-компонент, передаётся в `mdxComponents`

### `<Spoiler>`
- Клиентский компонент ("use client")
- `useState(false)` для toggle
- Скрытое состояние: кнопка "Показать ответ / Show answer" с иконкой глаза
- Раскрытое состояние: контент виден, кнопка меняется на "Скрыть ответ / Hide answer"
- Анимация: CSS transition на `opacity` + `grid-rows` (CSS grid trick для анимации высоты)

- Спойлер работает так же
